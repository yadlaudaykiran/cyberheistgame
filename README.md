# Cyberheist

A self-contained cyberpunk hacking simulation game for the browser. Players choose a contract, infiltrate a corporate grid, switch tools to bypass defenses, crack cipher locks, manage trace pressure, and escape with credits.

## Run

Open `index.html` in a modern browser.

No build step or package install is required.

## How to Play

- Choose a contract on the left.
- Start the breach.
- Move through adjacent network nodes.
- Use Spoof for relay nodes, Pulse for firewall nodes, and Decrypt for cipher or vault nodes.
- Cipher prompts show the reversed key. Enter the original four-digit key.
- Keep trace below 100% and reach the vault to finish the mission.
